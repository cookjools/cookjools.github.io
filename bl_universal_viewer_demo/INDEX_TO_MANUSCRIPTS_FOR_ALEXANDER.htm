Images

JPEG 2000 (jp2) images ingested into EAP are stored on the EAP IIIF server (AWS) under folder /EAP000/.

For example, the first image for EAP000_1_2 (i.e. add_ms_27261) is at https://images.eap.bl.uk/EAP000/EAP000_1_7/1.jp2/full/full/0/default.jpg .

The associated info.json file is at https://images.eap.bl.uk/EAP000/EAP000_1_7/1.jp2

 

Manifests

The IIIF manifests are created by a PHP script create_tei_facsimile_element_from_info_json_files.php. That script is run by Julian who then uploads the manifest onto his Github Pages using Github Desktop: for example, https://cookjools.github.io/bl_universal_viewer_demo/or_2265/script_created_manifest.json
